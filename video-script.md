## English Version

**[0:00-0:20] Interactive Narrative Visualization**
Hi, I'm presenting my Lab 8 submission. First, let me show you the interactive narrative visualization. As I scroll down the page, you can see the scatter plot updates dynamically, revealing commits progressively over time. Each commit appears with a narrative description, and the visualization smoothly animates to show only commits up to that point in time.

**[0:20-0:40] Unit Visualization**
Next, here's my unit visualization for file sizes. Each dot represents a line of code, and files are sorted by size in descending order. The colors represent different technologies - you can see CSS files in one color, JavaScript in another, and HTML in yet another. This gives a clear visual representation of the codebase composition.

**[0:40-1:00] Most Interesting Learning**

By combining Scrollama with D3's data binding, I can create visualizations that tell a story as users scroll, making data exploration more intuitive and memorable. The seamless integration of narrative text with animated visualizations creates a powerful storytelling experience.
